# Антология Всемирной литературы

Проект представляет собой цифровую антологию, в которой собраны русские переводы значимых памятников мировой литературы с древнейших времен до Нового времени. Основой для отбора текстов служит многотомная академическая «[История всемирной литературы](https://feb-web.ru/feb/ivl/)» (ИВЛ). Каждое произведение снабжено историко-литературным контекстом и публикуется с указанием первоисточника. Все переводы выполнены с помощью современных технологий искусственного интеллекта, что позволяет избежать правовых ограничений, связанных с авторскими правами, и делает тексты доступными для широкого круга читателей. Проект адресован всем, кто интересуется историей мировой литературы и стремится познакомиться с её ключевыми произведениями.
