# Белая Яджурведа. Книга I

*Перевод выполнен на основе английского перевода Ральфа Т.Х. Гриффита (1899), доступного на сайте sacred-texts.com. Книга I содержит ритуальные формулы и обращения к богам, преимущественно к Агни, для различных жертвоприношений.*

---

## Отдел 1

1. Для весеннего праздника (васанта) — в честь Агни. Для сезона дождей (варша) — в честь Индры. Для осеннего сезона (шарад) — в честь Всех Богов.

2. Для весны: двадцать одно жертвенное полено из дерева виканьката, и одно — из пальмы. Для сезона дождей: двадцать одно полено из дерева шми, и одно — из пальмы. Для осени: двадцать одно полено из дерева бильва, и одно — из пальмы.

3. Он кладёт (их) со словами: «О Агни, прими эти дрова! Пусть они принесут нам долгую жизнь и славу!».

4. Он кладёт (их), произнося имена богов: «Для Агни, для Индры, для Всех Богов».

5. Он кладёт дрова на алтарь и произносит: «О Агни, да будут эти дрова угодны тебе, как солнце угодно небу».

## Отдел 2

1. Он совершает жертвоприношение Агни, говоря: «Агни — первый из богов, он — жрец, он — владыка жертв».

2. Он разбрасывает священную траву (куша) вокруг алтаря со словами: «Пусть эта трава будет как ковёр для богов, как одеяние для земли».

3. Он возливает масло на жертвенный огонь и произносит: «Пусть это масло будет сладким для Агни, как мёд для пчелы».

4. Он приносит жертву из зерна и говорит: «Пусть это зерно будет пищей для богов, как хлеб для людей».

5. Он произносит молитву: «О Агни, прими эту жертву, даруй нам плодородие и победу над врагами».

6. Он возлагает жертву на огонь и говорит: «Пусть эта жертва достигнет небес, как птица достигает гнезда».

7. Он зажигает огонь и произносит: «О Агни, ты — свет жизни, ты — тепло мира. Мы приносим тебе дань, даруй нам благословение».

8. Он завершает первую часть обряда, говоря: «Да будет так. Да будет мир. Да будет мир. Да будет мир».

---

*Примечание: «Белая Яджурведа» (Śukla Yajurveda) представляет собой сборник прозаических и стихотворных формул, используемых жрецами-адхварью при совершении жертвоприношений. В отличие от «Чёрной Яджурведы», она содержит только сами ритуальные предписания без их теологического толкования, что делает её более лаконичной и ритуализированной.*
